## Description

This is a **NestJs** backend for a task management application. It has been created in learning purpose and help to experiment some concepts like :
>* Dynamic module (UserModule, ConfigurationModule)
>* Static module (TaskModule)
>* Middleware (AuthenticateMiddleware)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
